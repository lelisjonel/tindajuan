# TindaJuan Logo Assets

Generated logo files:

- `tindajuan-logo.svg` — full horizontal logo with wordmark and tagline
- `tindajuan-icon.svg` — app icon / favicon style logo

## Concept

**Storefront + TJ monogram + checkmark**

Meaning:

- Storefront = sari-sari store / tindahan
- TJ = TindaJuan
- Checkmark = recorded, organized, reliable
- Green = negosyo, pera, growth, trust
- Orange/yellow = local warmth and sari-sari store energy

## Colors Used

```txt
Primary Green: #16A34A
Dark Green:    #14532D
Warm Orange:   #F97316
Soft Yellow:   #FACC15
Background:    #FFFDF7
Muted Text:    #6B7280
```

## Suggested Usage

- Use `tindajuan-logo.svg` for landing page, docs, headers, and marketing.
- Use `tindajuan-icon.svg` for PWA icon, favicon, app tile, and social profile.

## Notes

These are editable SVG files, so they can be opened in Figma, Illustrator, Inkscape, or a browser.
